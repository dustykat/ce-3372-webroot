  • c) Using proprietary plugins for visual enhancements

6. What is the recommended contour interval for most hydrology-related maps?

  • a) 5 feet

  • b) 10 feet or 3 meters ✅

  • c) 20 feet

  • c) QGIS provides exclusive tools for water quality analysis

  • d) QGIS requires less user training

  • a) To show an elevation profile of the project area

  • b) To display alignments, existing features, and stationing of the project ✅

  • c) To illustrate subsurface utilities

  • d) To present 3D renderings of the project

  • a) SAGA terrain analysis ✅

  • b) Inverse distance interpolation

  • c) Raster-to-vector conversion

  • d) Linear regression models

10. Which element is typically included in the title block of a plan sheet?

  • a) Contour intervals

  • b) Township/Section/Range information

  • c) Project route and direction of flow ✅

  • d) Hydrological soil groups
